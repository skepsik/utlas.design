# Переводы (prose)

Выражения, которые в wiki **переводим** указанным образом. Список дополняй по мере необходимости.

**Не сюда:** идентификаторы кода (`` `runTurn` ``), EN compound блоком (*failure-driven fallback*, *tool loop*), enum в таблицах Wire (`omit`, `present`).

| EN | RU |
|----|-----|
| call site | вызывающий код |
| happy path | нормальный сценарий без abort |
| post-turn write | запись после `turn.stop` |
| north star | целевая архитектура (не § **Цель** с `[#N]`) |
| named strategy | именованная стратегия |
| business flow | обработка после LLM |
| cache | кеш (кеширование) |

Если выражения нет в таблице — по [README](./README.md) § Язык: переводим, если адекватен; иначе EN compound или термин как в коде.
